# AyijeCDM Extensions

AyijeCDM Extensions adds extra quality-of-life features to **AyijeCDM**.  
It is an add-on to Ayije CDM, not a replacement.

## Required Dependency

You must install **AyijeCDM** first:  
https://www.curseforge.com/wow/addons/ayijecdm

If Ayije CDM is not installed, this extension will not work.

## What This Extension Adds

- Adds `Move` and `Extensions` buttons inside the Ayije CDM settings window

![alt text](https://i.imgur.com/suEjU6s.png)
- Adds a simple move mode so you can drag CDM elements where you want them

![alt text](https://i.imgur.com/Pj2kHyO.jpeg)

- Adds a minimap button for quick access to Ayije CDM and extension settings

![alt text](https://i.imgur.com/DzrCLwA.png)

- Saves moved positions with CDM profiles

## What It Does Not Do

- It does not replace Ayije CDM
- It does not include Ayije CDM by itself
- It is not a standalone cooldown addon

## Installation

1. Install AyijeCDM (required)
2. Install AyijeCDM Extensions
3. Type `/reload` in game
